# PWD Contractor Collaboration Strategy (Odisha)

## 🎯 Exact Script

**Odia:**
“Sir, mu PWD Class B contractor achi. Mora license active achi. Jodi apanku billing, tender compliance ba license support darkar, ame collaborate kari pariba.”

**English:**
“I am a PWD Class B contractor with an active license. If you need support for billing, compliance, or license use, we can collaborate.”

---

## 💬 Follow-up Script

“Mu direct rent karuni, kintu partnership basis re kaam karuchi. Commission basis re 2–5% re deal kariba.”

---

## 💰 Commission Strategy

- Small work → 2–3%
- Medium work → 3–4%
- Risky work → 5%+

---

## 📍 Where to Find Work

1. PWD Offices (10 AM – 1 PM)
2. Construction Sites
3. Material Suppliers

## ⚡ 3-Day Action Plan
